Editorial Policy

This site aggregates and indexes content from reputable primary/official sources. We do not scrape HTML or republish full articles. Views belong to the original publishers. If you are a publisher and want inclusion or removal, please open an issue in the repository.

How it works

A scheduled GitHub Action pulls data via public APIs and writes data/*.json. The frontend performs local full‑text search with MiniSearch and renders citation cards that link back to source pages.
